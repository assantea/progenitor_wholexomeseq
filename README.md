# Whole Exome Sequencing Analysis of Bronchial Dysplasia Samples

## Overview

This repository contains code and quality control summaries used
for whole exome sequencing analyses of bronchial dysplasia samples.

Analyses include:

- Variant filtering
- VAF analysis
- Mutation burden
- CNV analysis
- Oncoplot generation

## Software

nf-core/sarek v3.8.1

Reference genome:
GRCh38
